## Employee REST API With Newman
## About this project
This project demonstrates API testing of an Employee REST API using Postman. It includes multiple test cases covering positive and negative scenarios, along with automated execution using Newman and detailed test reports.
## Prerequisites
- Postman
- Node.js & npm
- Newman
- Newman HTML Reporter
